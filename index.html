CSS selectors

What is a selector?

Selector defines which CSS rule is to apply to which element.
Note that there is no way to choose a parent of the element or children of parents' neighbour.

Most used selectors

Selector Description Example
Type selectors This selector is used to match all elements of a given type or tag name. h1 {
color: red;
}
Class selectors These selectors are considered to be more specific compared to type selectors.
For selecting elements having heading as the value of the class attribute, a . needs to be prepended. 		.heading {
font-weight: bold;
}
ID selectors ID selectors are most specific of all. Note that ID can be used only once per project.
For selecting the element having time as the value of the id attribute, a # needs to be prepended.		 #time {
font-family: Arial;
}

Combining selectors

Name Explanation Example
Chaining selectors CSS selectors can be chained so that rule sets apply only to elements that match all criteria.
For example, here <h1> elements with .heading class will be teal.		 h1.heading {
color: teal;
}
CSS descendant selector The CSS descendant selector combinator is used to match elements that are descended from another matched selector.
They are denoted by a single space between each selector and the descended selector.
Note that all matching elements are selected regardless of the nesting level in the HTML.		 body p {
text-align: center;
background-color: blue;
font-size: 10px;
}
Groups of selectors Multiple selectors can be matched to the same rule using comma.
For example, here <h1>, <h2> and <td> elements would be bold.		 h1, h2, td {
font-weight: bold;
}
